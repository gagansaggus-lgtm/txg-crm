How it runs

Three layers. One operating system.

The team works in TXG Vector. Claude Code runs as a scheduled agent — generating content, enriching leads, drafting outreach — authenticated through the existing Claude Max plan. Supabase is the persistent state shared by both. Delivery happens through inexpensive pipes (Resend for email, Ayrshare for social, WhatsApp Business API).
